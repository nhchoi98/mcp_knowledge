# Layout

# Form

### **Horizontal Form Layout**

(한국어로는 “**가로형 폼 레이아웃**” 또는 “**수평 정렬 폼**”)

이게 가장 통용되는 표현이에요.

즉, `label`과 `input`이 같은 행(row)에 가로로 나란히 배치된 구조를 말합니다.

| 형식 | 설명 |
| --- | --- |
| **Vertical Form Layout** | label이 위, input이 아래로 쌓이는 형태 (모바일에서 흔함) |
| **Inline Form Layout** | label과 input이 한 줄에 있으면서 여러 필드가 한 줄에 연속 배치됨 (검색바, 필터 UI 등에서 자주 사용) |
| **Two-column Form / Split Form** | label과 input을 각각 왼쪽/오른쪽 칼럼처럼 분리한 폼 (대형 엔터프라이즈 UI에서 흔함) |
