## 프로젝트 규모별 가이드

### 소규모 프로젝트 (MVP, 스타트업 초기)

```
✅ 3NF까지만
✅ 명확한 외래키 관계
✅ 기본 인덱스
❌ 과도한 최적화 No
❌ 미리 비정규화 No

```

### 중규모 프로젝트 (성장 단계)

```
✅ 3NF 기본
✅ 성능 모니터링
✅ 병목 지점 발견 시 선택적 비정규화
✅ 캐싱 레이어 도입
✅ 읽기 전용 복제본 고려

```

### 대규모 프로젝트 (엔터프라이즈)

```
✅ 3NF + 선택적 비정규화
✅ 파티셔닝, 샤딩
✅ CQRS 패턴 (읽기/쓰기 분리)
✅ 이벤트 소싱
✅ 폴리글랏 퍼시스턴스 (SQL + NoSQL)
```
